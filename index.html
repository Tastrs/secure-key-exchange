What is Babel?

Babel is an Open Source widget and API built with the PubNub Realtime network, that let's you exchange 1024-bit RSA Public Keys and send Encrypted, Self-Destructing Messages.

How do I use Babel?

First, import the necessary JS libraries. Babel requires PubNub, jQuqery, and Cryptico. 
<script src="https://cdn.pubnub.com/pubnub.min.js"></script>
<script src="http://code.jquery.com/jquery-2.1.1.min.js"></script>
<script src="https://rawgit.com/lw7360/babel/gh-pages/js/libs/cryptico.js"></script>
And initialize Babel with a username that other users will be able to identify you with. 
var babel = new babel('doge');

For Public Key Exchange

Babel uses 1024-bit RSA encryption to encrypt the messages it sends. However, if you would like, you could use these keys for other purposes. 
listUsers: function() {
    // Returns a JS object with User Names as members and Public Keys as their values.
    // e.g. {"doge" : "much_public_key", "Alice" : "Alice's_public_key"};
},
mySelf: function() {
    // Returns your User Name and Public Key
    // e.g. {username: "doge", public_key: "much_public_key"};
},
getPubKey: function(username) {
    // Returns the Public Key of username. If username isn't subscribed to the channel, this function returns null.
    // e.g. exchange.getPubKey("doge") === "much_public_key"
    // exchange.getPubKey("wowe") === null
},
quit: function() {
    // Doesn't return anything. After quitting, other users can no longer get your Public Key.
}

For Encrypted, Self-Destructing Messages

For key-exchange and self-destructing messages, you can use dogechat.js. 
var doge = new dogechat("doge");
Doge Chat generates it's own 1024 bit RSA key for encryption, and has a couple publicly accessible methods.
