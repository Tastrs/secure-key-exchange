What is DogeChat?

DogeChat is an Open Source widget and API built with the PubNub Realtime network. With DogeChat's API you can exchange Public Keys and also send Encrypted Self-Destructing Messages.

How do I use DogeChat?

Public Key Exchange

If all you need to do is perform key-exchange between two web clients, you should use dogekeyexchange.js.
Supply a username, application name, and public key when you initialize a DogeKeyExchange instance. 
var exchange = DogeKeyExchange("doge", "doge_chat", "much_public_key");
DogeKeyExchange has four publicly accessible methods. 
listUsers: function() {
    // Returns a JS object with User Names as members and Public Keys as their values.
    // e.g. {"doge" : "much_public_key", "Alice" : "Alice's_public_key"};
},
mySelf: function() {
    // Returns your User Name and Public Key
    // e.g. {username: "doge", public_key: "much_public_key"};
},
getPubKey: function(username) {
    // Returns the Public Key of username. If username isn't subscribed to the channel, this function returns null.
    // e.g. exchange.getPubKey("doge") === "much_public_key"
    // exchange.getPubKey("wowe") === null
},
quit: function() {
    // Doesn't return anything. After quitting, other users can no longer get your Public Key.
}

Self-Destructing Messages

For key-exchange and self-destructing messages, you can use dogechat.js. 
var doge = new dogechat("doge");
Doge Chat generates it's own 1024 bit RSA key for encryption, and has a couple publicly accessible methods. 
sendMessage: function(recipient, message, ttl) {
    // Sends your message to the recipient with a ttl (time to live).
},
onRecieveMessage: function(callback) {
    // Change the function that's called when you get a new message in Doge Chat.
},
onPresence: function(callback) {
    // Change the function that's called when a user connects or disconnects into Doge Chat.
},
listUsers: function() {
    // Return the User Names of all users and their associated public keys.
},
returnMessages: function() {
    // Returns all Messages sent to you or sent by you, that haven't expired yet.
},
deleteMessage: function(userName, msgID) {
    // Deletes the message with ID === msgID in the conversation with userName.
},
myKey: function() {
    // Returns your RSA key generated by Cryptico.
}
